# 🔐 Password Generator

A modern, customizable password generator built with React, Vite, and Tailwind CSS. Generate strong, secure passwords with various options.

## ✨ Features

- **Password Length** (4-50 characters)
- **Character Types:**
  - ✅ Uppercase Letters (A-Z)
  - ✅ Lowercase Letters (a-z)
  - ✅ Numbers (0-9)
  - ✅ Special Characters (!@#$%^&*)
- **Real-time password generation**
- **Copy to clipboard** functionality
- **Password strength indicator**
- **Responsive design**

- **Tailwind CSS** - Styling
